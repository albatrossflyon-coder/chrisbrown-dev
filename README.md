# chrisbrown-dev

Personal developer portfolio — a scroll-driven showcase of 5 real projects (Skinstric, vuln-hunter, AI Infrastructure (RAG + Omni Dashboard), agentic-rust-mcp, job-lead-discovery), built with Next.js, React Three Fiber, and GSAP ScrollTrigger.

See `BUILDLOG.md` for session-by-session build history and `DECISIONS.md` for architecture/design rationale.

## Development

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).
